# 喵 (Mao) · 发布通道

一只浮在 macOS 桌面上的 Live2D 伙伴 —— 有脸、会陪你、记得住你，还能连上你的 App 和网页帮你干活。默认全在本地。

本仓库是「喵」的**公开发布通道**，只托管两样东西：

- **安装包** —— 见 [Releases](https://github.com/solaswing/avatar-releases/releases/latest)（已 Developer ID 签名 + Apple 公证，干净 Mac 双击即用，无 Gatekeeper 警告）。
- **更新清单** —— [`latest.json`](latest.json)，App 内「检查更新」比对构建号用。

源码仓库另置，不在此处。

macOS 14+ · Apple Silicon
